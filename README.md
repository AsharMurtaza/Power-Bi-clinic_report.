Project Overview:
This project is an interactive Clinic Performance Report built using Microsoft Power BI.
It provides a comprehensive view of patient flow, demographics, and departmental performance, helping clinics monitor operations and optimize patient management.

Key Metrics:
Total Number of Patients
In-Patients & Out-Patients Count
Daily Average Patients
In-Patient Percentage
Average Patient Age

Data Analysis & Visualizations:
The report includes the following analyses:
Patient Count by Date (Trend Analysis)
Patient Count by Department
Patient Count by Gender
Patient Distribution by Age Groups (Age Bins)
Doctor-wise Patient Count

Insights Generated:
Identified peak patient days
Compared department performance
Analyzed gender distribution
Evaluated age group trends
Measured doctor workload distribution

Tools Used:
Microsoft Power BI
Data Modeling & Transformation
DAX Calculations
Interactive Data Visualizations
